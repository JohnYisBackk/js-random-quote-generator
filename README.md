# Random Quote Generator 🎲

Vanilla JavaScript random quote generator with smooth animations, clipboard copy
and localStorage support.

## Live Demo 🚀
👉 https://YOUR_USERNAME.github.io/js-random-quote-generator/

## Features
- Random quote generation
- Prevents the same quote from showing twice in a row
- Fade-out / fade-in animations
- Shine gradient text effect
- Copy quote to clipboard
- Last quote saved in localStorage
- Fully responsive (mobile-friendly)

## Built with
- HTML
- CSS
- Vanilla JavaScript

## What I learned
- Working with arrays and random logic
- DOM manipulation and event handling
- CSS animations and transitions
- Clipboard API
- localStorage (save & load state)
- Improving UX with small details and animations

## Project purpose
This project was built as a learning exercise to practice JavaScript fundamentals
and UI polish without using any frameworks.
